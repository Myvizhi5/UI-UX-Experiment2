# UI-UX-Experiment2

## Aim:

To design a basic mobile and web layout for a case study using Gestalt principles, UI elements, usability principles, interaction behaviors, and a brand logo created in Canva.

## Algorithm:

1. Select an existing case study and identify user requirements.
2. Plan and create mobile and web layout wireframes.
3. Apply Gestalt principles such as proximity, similarity, and continuity in the design.
4. Add UI elements and design patterns like buttons, menus, forms, and navigation bars.
5. Develop interaction behaviors and usability features for smooth user experience.
6. Create a brand logo using Canva and integrate it into the layouts.
